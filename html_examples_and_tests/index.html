Groove Scribe — Examples & Test Pages

Standalone example, demo, and test pages, moved out of the project root into this folder. The main application lives at ../index.html. Serve the project over HTTP (npm run serve) rather than opening these via file://.

Examples & demos

  • GScribeMusicImageOnly.html

    Renders a groove as a standalone music-notation image, with no editor UI.

  • GrooveMultiDisplay.html

    Demonstrates rendering several read-only GrooveDisplay instances on a single page.

  • GrooveEmbedSingle.html

    A single-groove display variant (companion to the production ../GrooveEmbed.html embed page).

  • InterestingGrooves.html

    A curated catalog of grooves (rock, shuffle, jazz, latin, …); each entry links into the main editor with that groove loaded.

  • GrooveDBCreateGroove.html

    Authoring tool for building groove entries for the groove database.

Test pages

  • TestAllTimeSigs.html

    Writes out a hi-hat groove in every time signature for a chosen subdivision — a visual check of time-signature handling.

  • WeirdTimeSigsTest.html

    Renders a battery of unusual / edge-case time signatures for inspection.

  • GrooveEmbedTest.html

    Embeds the production ../GrooveEmbed.html in an iframe to verify embedded rendering.

  • grooveDBTest.html

    Exercises the GrooveDB "put groove on page" formatting path (GrooveDBFormatPutGrooveOnPage).

  • grooveDBTest - Single.html

    Single-groove variant of the GrooveDB formatting test.
